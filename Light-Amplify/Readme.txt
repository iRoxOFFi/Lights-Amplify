FR :
Une fois le fichier téléchargé, suivez ces étapes :

1. Rendez-vous à l'emplacement du fichier FiveM.

2. Une fois à l'intérieur, accédez à "FiveM Application Data".

3. Allez dans "citizen".

4. Allez dans "common".

5. Allez dans "data".

6. Ajoutez le fichier "visualsettings.dat" dans le dossier "data".

----------------------------------

EN : 
Once the file is downloaded, follow these steps:

1. Go to the location of the FiveM file.

2. Once inside, navigate to "FiveM Application Data."

3. Go to "citizen."

4. Go to "common."

5. Go to "data."

6. Add the file "visualsettings.dat" to the "data" folder.
